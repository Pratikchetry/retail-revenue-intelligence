# Retail Revenue Intelligence & Anomaly Detection

This project is being built to create a retail analytics system that helps understand sales performance, identify unusual revenue patterns, and support customer segmentation from transaction data.

## Current Status
So far, the project has completed:
- raw data understanding
- data cleaning
- exploratory data analysis (EDA)

## Dataset
The project currently uses the **Online Retail II** dataset as the core transaction source.

## Tools Planned / In Progress
- **Python** for cleaning, analysis, and ML
- **PostgreSQL** as the central data layer
- **Tableau** for dashboards
- **Power BI / DAX** as a companion BI showcase

## Project Workflow
1. Data understanding
2. Data cleaning
3. Exploratory data analysis
4. PostgreSQL schema and loading
5. RFM segmentation
6. Anomaly detection
7. Forecasting
8. Validation
9. Tableau and Power BI integration

## Notes
This repository is being built step by step. Final business results, dashboard screenshots, DAX outputs, and ML results will be added after the remaining project stages are completed.
